# Inteligência Artificial Generativa Aplicada à Programação - SENAI

Este é o repositório central destinado ao armazenamento de todos os projetos, exercícios e códigos práticos desenvolvidos durante o curso de **Inteligência Artificial Generativa Aplicada à Programação** realizado no SENAI. 

O objetivo do espaço é consolidar o aprendizado em lógica de programação, desenvolvimento de software e a integração de ferramentas de IA para otimização e automação de código.

## 🚀 Tecnologias e Ferramentas

* **Python** como linguagem principal de programação.
* Ferramentas de IA Generativa para assistência no desenvolvimento.

## 📂 Estrutura do Repositório

O repositório é atualizado continuamente e está organizado da seguinte forma:

* **Aulas Práticas**: Códigos e conceitos abordados em ambiente de sala de aula.
* **Exercícios de Fixação**: Desafios e problemas propostos para consolidar a lógica de programação.
* **Projetos Autônomos**: Pequenas aplicações desenvolvidas para aplicar os conhecimentos integrados de programação e IA.

## 🛠️ Como Executar os Arquivos

1. **Clone este repositório:**
   ```bash
   git clone https://github.com/fxxggah/vibecode-senai
   ```

2. **Navegue até a pasta do arquivo que deseja executar:**
   ```bash
   cd vibecode-senai
   ```

3. **Execute o script utilizando o interpretador Python:**
   ```bash
   py nome_do_arquivo.py
   ```
